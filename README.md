# Vulkan Guide - C# Port

This repository is a C# port of the original [Vulkan Guide](https://github.com/vblanco20-1/vulkan-guide). It provides both Version 1 and Version 2 of the guide, with each version maintained in a separate branch.

## Branches
- **[v1 Branch](https://github.com/GodOfMemes/VkGuideCSharp/tree/v1)** - C# implementation of Vulkan Guide V1.
- **[v2 Branch](https://github.com/GodOfMemes/VkGuideCSharp/tree/main)** - C# implementation of Vulkan Guide V2 (latest).
